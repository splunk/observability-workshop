## Introduction

Gain hands-on experience with Splunk Observability Cloud using the OpenTelemetry Demo (Astronomy Shop) v2.0.1 application. This modular workshop is designed so trainers can pick and choose which product modules to deliver based on audience needs and available time.

## How This Workshop Works

Every session starts with two shared setup chapters:

1. **Login** — Access Splunk Observability Cloud (5 min)
2. **The Astronomy Shop** — Explore the demo application and generate telemetry data (10 min)

Then the trainer selects any combination of the following standalone modules:

| Module | Duration | Description |
|--------|----------|-------------|
| **Log Observer** | 20 min | Use Log Observer as a primary investigation tool to triage issues using filtering, grouping, and pattern analysis |
| **APM** | 20 min | Trace requests across microservices and identify performance bottlenecks *(coming soon)* |
| **DEM** | 25 min | Digital Experience Monitoring — Real User Monitoring and Synthetics *(coming soon)* |
| **DB-MON** | 20 min | Database Monitoring — investigate slow queries and correlate with application health *(coming soon)* |
| **Secure APP** | 20 min | Application Security — detect and investigate vulnerabilities in running applications *(coming soon)* |
| **Infrastructure Monitoring** | 20 min | Monitor hosts, containers, and Kubernetes clusters *(coming soon)* |

Each module is self-contained after the shared setup — no module depends on another.
