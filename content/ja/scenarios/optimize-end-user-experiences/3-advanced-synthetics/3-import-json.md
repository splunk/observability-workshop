テストの設定を開始するには、Chrome DevTools Recorder からエクスポートした JSON をインポートする必要があります。{{% button %}}**Import**{{% /button %}} ボタンを有効にするには、まずテストに名前を付ける必要があります（例: **[`<your team name>`] `<your initials>` - Online Boutique**）。

![Browser edit form](../_img/browser-form.png)

{{% button %}}**Import**{{% /button %}} ボタンが有効になったら、それをクリックして、Chrome DevTools Recorder からエクスポートした JSON ファイルをドロップするか、ファイルをアップロードします。

![Import JSON](../_img/browser-import.png)

JSON ファイルがアップロードされたら、{{% button style="blue" %}}Continue to edit steps{{% /button %}} をクリックします。

![Import complete message](../_img/browser-import-complete.png)

テストを編集する前に、まず設定を構成しましょう。{{% button style="blue" %}}< Return to test{{% /button %}} をクリックします。

![Return to test button in the browser test editor](../_img/browser-return.png)
