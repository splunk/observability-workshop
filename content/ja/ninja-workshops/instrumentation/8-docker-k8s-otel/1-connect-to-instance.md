## EC2 インスタンスに接続する

各参加者用に AWS/EC2 に Ubuntu Linux インスタンスを用意しています。

講師から提供された IP アドレスとパスワードを使用して、以下のいずれかの方法で EC2 インスタンスに接続してください:

* Mac OS / Linux
  * ssh splunk@IP address
* Windows 10+
  * OpenSSH クライアントを使用します
* それ以前のバージョンの Windows
  * Putty を使用します
