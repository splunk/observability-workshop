このワークショップでは、以下の概念についてハンズオン形式で学びました。

* **Splunk Distribution of the OpenTelemetry Collector** を Linux ホストにデプロイする方法。
* .NET アプリケーションを **Splunk Distribution of OpenTelemetry .NET** で計装する方法。
* .NET アプリケーションを「Docker 化」し、**Splunk Distribution of OpenTelemetry .NET** で計装する方法。
* Helm を使用して **Splunk Distribution of the OpenTelemetry Collector** を Kubernetes クラスターにデプロイする方法。
* Collector の構成をカスタマイズし、問題をトラブルシュートする方法。

OpenTelemetry を使用したその他の言語や環境の計装方法を確認するには、
[Splunk OpenTelemetry Examples GitHub リポジトリ](https://github.com/signalfx/splunk-opentelemetry-examples) を参照してください。

今後、このワークショップを自分自身で実施する場合は、本手順を参照しつつ、Splunk Show の **Splunk4Rookies - Observability**
ワークショップテンプレートを使用して EC2 インスタンスをプロビジョニングしてください。
