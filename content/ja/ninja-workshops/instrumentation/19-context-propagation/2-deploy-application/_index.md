このフェーズでは、アプリケーションが動作し、API リクエストが有効なエンドツーエンドのレスポンスを返すことを確認します。

このセクションの終わりまでに、以下の状態になります:

1. 計装なしで動作する Python マイクロサービス
2. アプリケーションがリクエストを受信し、出力データが有効であることの検証
3. この段階では Splunk Observability Cloud にデータが存在しないことの確認
