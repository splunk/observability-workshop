## EC2インスタンスに接続する

各参加者用にAWS/EC2上にUbuntu Linuxインスタンスを用意しています。

講師から提供されたIPアドレスとパスワードを使用して、以下のいずれかの方法でEC2インスタンスに接続します。

* Mac OS / Linux
  * ssh splunk@IPアドレス
* Windows 10+
  * OpenSSHクライアントを使用します
* 以前のバージョンのWindows
  * Puttyを使用します
