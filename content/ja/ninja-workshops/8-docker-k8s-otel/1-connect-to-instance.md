## EC2 インスタンスへ接続

各参加者のために、AWS/EC2にUbuntu Linuxインスタンスを用意しました。

インストラクターから提供されたIPアドレスとパスワードを使用して、以下のいずれかの方法でEC2インスタンスに接続してください

- Mac OS / Linux
  - ssh splunk@IPアドレス
- Windows 10+
  - OpenSSHクライアントを使用
- 以前のバージョンのWindows
  - Puttyを使用
