このフェーズでは、シンプルな Java アプリケーションをビルドし、AppDynamics Java Agent をアタッチして、APM データが AppDynamics Controller に表示されることを確認します。

これは、現在ほとんどの AppDynamics ユーザーが使用している「通常の」単一送信先インストルメンテーションです。
