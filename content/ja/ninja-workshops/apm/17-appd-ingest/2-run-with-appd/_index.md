このフェーズでは、シンプルなJavaアプリケーションをビルドし、AppDynamics Java Agentをアタッチして、APMデータがAppDynamics Controllerに表示されることを確認します。

これは、現在ほとんどのAppDynamicsのお客様が利用している「通常」の単一宛先への計装方法です。
