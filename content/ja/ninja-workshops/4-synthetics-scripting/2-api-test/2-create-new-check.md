## 新しい API テストを作成

{{< button style="blue" >}}Add new test{{< /button >}} ボタンをクリックし、ドロップダウンから **API test** を選択して新しいAPIテストを作成します。テスト名には **イニシャル** に続けて **Spotify API** と入力します（例: **RWC - Spotify API**）。
