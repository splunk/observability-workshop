Neste workshop, demonstraremos como o Splunk Observability Cloud oferece visibilidade instantânea da experiência do usuário, abrangendo tudo, desde aplicativos front-end até serviços back-end. Você terá a oportunidade de explorar alguns dos recursos mais poderosos da plataforma, que a diferenciam de outras soluções de observabilidade:

- **Monitoriação de Infrasestrutura**
- **Full-fidelity Real User Monitoring (RUM)**
- **Visibilidade completa de traces de ponta a ponta com NoSample Full-fidelity Application Performance Monitoring (APM)**
- **Consulta de log sem código**
- **Testes sinteticos**
- **Análise de causa raiz com análise de tags e stack traces**
- **Related Content (Conteudo Relacionado)** para navegação perfeita entre componentes

Um dos principais pontos fortes do Splunk Observability Cloud é sua capacidade de unificar dados de telemetria, criando uma imagem abrangente da experiência do usuário final e de toda a sua pilha de aplicativos.

O workshop se concentrará em um aplicativo de comércio eletrônico baseado em microsserviços implantado em instâncias AWS EC2. Os usuários podem navegar pelos produtos, adicionar itens ao carrinho e concluir compras. Este aplicativo é totalmente instrumentado com OpenTelemetry para capturar dados detalhados de desempenho.

**O que é OpenTelemetry?**
OpenTelemetry é uma coleção de ferramentas, APIs e kits de desenvolvimento de software (SDKs) de código aberto projetados para ajudá-lo a instrumentar, gerar, coletar e exportar dados de telemetria, como métricas, rastreamentos e logs. Esses dados permitem uma análise aprofundada do desempenho e comportamento do seu software.

A comunidade OpenTelemetry está crescendo rapidamente, apoiada por empresas líderes como Splunk, Google, Microsoft e Amazon. Atualmente tem o segundo maior número de contribuidores dentro da Cloud Native Computing Foundation, atrás apenas do Kubernetes.
