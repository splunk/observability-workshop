{{% notice icon="user" style="orange" title="Persona" %}}

Você é um **engenheiro de front-end** ou um **SRE** encarregado de fazer a primeira triagem de um problema de desempenho. Você foi solicitado a investigar um possível problema de satisfação do cliente com o aplicativo Online Boutique.

{{% /notice %}}

Examinaremos os dados reais do usuário fornecidos pela telemetria recebida das sessões de navegador de todos os participantes. O objetivo é encontrar uma sessão de navegador, celular ou tablet com desempenho insatisfatório e iniciar o processo de solução de problemas.
